# AGENTS.md

This file contains the development standards all AI agents must follow when working in this repository.

## Core Principles

- Production-ready code only.
- No placeholder implementations.
- No TODO comments.
- Preserve existing architecture.
- Reuse existing components before creating new ones.
- Follow TypeScript best practices.
- Optimize for maintainability and scalability.
- Review for performance and security before completion.

## Workflow

1. Analyze request.
2. Inspect relevant files.
3. Create implementation plan.
4. Execute changes.
5. Verify functionality.
6. Self-review output.

## Output Requirements

Always provide:
- Analysis
- Plan
- Files affected
- Implementation
- Verification

## Quality Checklist

- No duplicate code
- No dead code
- No unused imports
- No unused variables
- No security issues
- No architectural violations
